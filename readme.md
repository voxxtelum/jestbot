## Config.json

1. Add config.json file with oauth token from Discord
2. Add bot's User ID
3. Prefix to get bots attention

```json
{
    "token"  : "_Paste bot token ID here from Discord Developer site_",
    "botID"  : "_Paste bot's User ID_",
    "prefix" : "!"
}
```
